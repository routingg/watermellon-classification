# 프로젝트 설명:
- 40분 분량의 수박을 두드린 소리 수집
- 1초단위로 나누고, 소리가 빈 부분은 자르기
- 1초길이의 음성파일 2404개 수집(익은 수박 1202개, 덜익은 수박 1202개)
- melspectrogram을 특징으로 잡아 학습 진행
